Easy Integration

Add Smart Chat to your website in under 2 minutes:

Option 1: Auto-initialization with script tag

<script src="/chatbot-widget.js" 
        data-auto-init="true"
        data-title="Your Chat Assistant"
        data-position="bottom-right"
        data-api-url="/api/chat/ask"></script>

Option 2: Manual initialization with JavaScript

<script src="/chatbot-widget.js"></script>
<script>
    const chatbot = new ChatbotWidget({
        title: 'Your Chat Assistant',
        position: 'bottom-right',
        apiUrl: '/api/chat/ask',
        placeholder: 'Ask me anything...'
    });
</script>

Configuration Options

{
    apiUrl: '/api/chat/ask',      // Your chatbot API endpoint
    title: 'Smart Chat',          // Widget title
    position: 'bottom-right',     // 'bottom-right' or 'bottom-left'
    placeholder: 'Type here...',  // Input placeholder text
    theme: 'light'                // Theme (future feature)
}
